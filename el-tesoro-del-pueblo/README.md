# 🎬 El Tesoro del Pueblo

> Aprende Bitcoin con las estrellas de la Época de Oro del cine mexicano

**Hackathon Bitcoin México 2026**

🌐 **Demo en vivo:** https://el-tesoro-del-pueblo.vercel.app

---

## ¿Qué es?

Una novela visual educativa con mini-juegos que enseña los conceptos fundamentales de Bitcoin a través de los personajes icónicos del cine mexicano de los años 40 y 50.

## Personajes

- 🎸 **Pedro Infante** — ¿Por qué Bitcoin es mejor que el banco?
- 💎 **María Félix** — La llave privada y la custodia
- 🎩 **Cantinflas** — Cómo funciona una transacción
- 🤠 **Jorge Negrete** — Bitcoin vs. inflación
- 🌹 **Dolores del Río** — Remesas sin fronteras
- 🏦 **El Cacique** *(villano)* — El sistema financiero tradicional

## Mini-juegos

| Acto | Juego | Concepto |
|---|---|---|
| I | 🤠 Atrapa los Sats | Custodia propia |
| II | 🪀 El Balero | Volatilidad |
| III | 🔐 La Caja Fuerte | Llave privada |
| IV | 🎩 El Enredo de Cantinflas | Transacciones |
| V | ⏳ La Carrera del Tiempo | Inflación |
| VI | 🗺️ Traza la Ruta | Remesas Lightning |

## Tecnología

- HTML + CSS + JavaScript vanilla (sin dependencias)
- Web Audio API para efectos de sonido
- Lightning Network para reclamo de sats
- Deploy en Vercel

## Cómo correr localmente

```bash
npx serve .
```
